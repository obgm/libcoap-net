◆ coap_add_option_internal()

size_t coap_add_option_internal ( coap_pdu_t *  pdu,
coap_option_num_t  number,
size_t  len,
const uint8_t *  data 
)

Adds option of given number to pdu that is passed as first parameter.

The internal version of coap_add_option() may cause an option to be inserted, even if there is any data in the pdu.

Note: Where possible, the option data needs to be stripped of leading zeros (big endian) to reduce the amount of data needed in the PDU, as well as in some cases the maximum data size of an option can be exceeded if not stripped and hence be illegal. This is done by using coap_encode_var_safe() or coap_encode_var_safe8().

Parameters
pduThe PDU where the option is to be added.
numberThe number of the new option.
lenThe length of the new option.
dataThe data of the new option.
Returns
The overall length of the option or 0 on failure.
